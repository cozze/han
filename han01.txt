로컬 저장소

 git  관리 3  나무



첫번 나무-

작업 디렉토리(Working directory)
실제 파일,


 두번째 나무-

 인덱스(Index)

준비 영역(staging area)의 역할


마지막 나무-

 HEAD

최종 확정본(commit). 



----
변경된 파일은 아래 명령어로 (인덱스에) 추가.
git add <파일 이름>
git add *
 git의 기본 작업 흐름에서 첫 단계


-----

 실제로 변경 내용을 확정하려면
  명령을 내려야 한다.
git commit -m " 확정본 예증"

변경 파일 HEAD에 반영.


원격 저장소에는 아직 반영 안됨.
